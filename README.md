# Traktor Organizer

> A smarter way to browse, tag, and curate your Traktor DJ collection.

Traktor Organizer is a desktop app that reads your Traktor collection and gives you a fast, flexible way to organize your tracks, build playlists, and export them back into Traktor. Built for DJs with large collections who find Traktor's built-in tools limiting.

## How It Works

1. Import your `collection.nml` from Traktor
2. Browse, search, and tag your tracks in a fast table view
3. Build playlists by combining tags, key, genre, and rating filters
4. Export playlists as `.nml` files — import them directly into Traktor Pro

## Capabilities

### Library Browser

- Full track table with all Traktor metadata: title, artist, BPM, key, genre, rating, label, and more
- Inline editing for title, artist, genre, star rating, and tags
- Resizable, reorderable, and hideable columns — customize what you see
- Cover art thumbnails pulled from your audio files

### Tags

- Tags are auto-extracted from Traktor's Comments field (space-separated words)
- Tag Cloud sidebar shows every tag in your collection with usage counts
- Click one or more tags to filter instantly — multiple tags use AND logic
- Blocklist any tag to hide it everywhere and remove it from all tracks automatically

### Filters

- Global search across title, artist, album, genre, and tags at once
- Filter by musical key (Open Key / Camelot notation), genre, or minimum star rating
- All filters stack — combine them to drill down to exactly what you need

### Playlists

- Save any filtered view as a named playlist
- Edit playlists: remove tracks, make inline edits, apply additional filters
- Smart drift detection: if your saved filters would now return different tracks, you're notified and can update the playlist in one click
- Export any playlist as a Traktor-compatible `.nml` file ready to import

### Audio Preview

- Built-in mini player — preview any track without leaving the app
- Seek through tracks, skip forward/back, with cover art display
- Smooth crossfades between tracks

## A Few Things to Know

- Your `collection.nml` is **never modified** — all exports are separate files
- Re-import your collection anytime; tags, ratings, and genre edits you've made are preserved
- All data stays on your machine — local database, no cloud, no accounts required
- AIFF files are not currently supported for audio preview
